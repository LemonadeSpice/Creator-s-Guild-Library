## Default Hotkeys

| Combination | Effect                         |
| ----------- | ------------------------------ |
| Ctrl + K    | Turn selected text into a link |
| Ctrl + L    | Toggle checkbox                |
## Custom hotkeys

| Combination                          | Effect               |
| ------------------------------------ | -------------------- |
| Ctrl + (the "+" not from the numpad) | Toggle bullet list   |
| Ctrl + {                             | Toggle numbered list |
| Alt + up arrow                       | Move line up         |
| Alt + down arrow                     | Move line down       |
| Ctrl + -                             | Toggle code line     |
| Ctrl + Shift + -                     | Toggle code block    |
| Ctrl + Q                             | Git pull             |
| Ctrl + A                             | Git commit & sync    |
