## As a contributor
1. Go to your GitHub repository's settings, Collaborators page ![[Pasted image 20260614151837.png]]
2. Add any collaborators ![[Pasted image 20260614151932.png]]
3. Tell them to follow the rest of the guide

### Prerequisites
- A [GitHub](https://github.com/) account
- [Github Desktop](https://desktop.github.com/download/) (simplest local git login cause fuck the new buggy console login)
- [Obsidian](https://obsidian.md/)
- If you want to preview the site locally you will also need, otherwise GitHub will handle making the web
	- [nodejs](https://nodejs.org/en)
### Steps to get editing
1. Tell the host to add you as a collaborator to the repository (guide right above)
2. Open Github Desktop
	1. Click on the repository selection dropdown ![[Pasted image 20260614101850.png]]
	2. Select Clone repository, it should display all repositories you have access to ![[Pasted image 20260614101808.png]]
3. Once the repository is local, just open it on Obsidian by using the second option here ![[Pasted image 20260614104009.png]]
	- You can quickly navigate to where your repository is by right clicking and using Show in Explorer ![[Pasted image 20260614152624.png]]
	- **IMPORTANT:** The vault will only save by itself locally, you manually need to commit and push changes, refer to [[How to update the library]]
