## Saving
Obsidian handles saving on its own, don't worry about it

## Git usage from Obsidian
- Ctrl + P to open the obsidian command bar
- Type "Git"
	- Git commit
		- Rounds up all local changes into a "commit"
	- Git: push
		- Uploads all commits to "origin" (the online repository)
	- Git: pull
		- Downloads all commits

## Handling of conflicts
Ideally, just don't edit files at the same time

If your local version and the online version ("origin" or "remote" in Obsidian) have discrepancies with a file, a merge conflict will happen, same with doing pull requests if enabled

[Github Desktop](https://desktop.github.com/download/) can fix most conflicts, otherwise use console commands
