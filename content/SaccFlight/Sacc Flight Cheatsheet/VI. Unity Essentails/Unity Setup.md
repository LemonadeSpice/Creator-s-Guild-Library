# Prerequisite 
a couple of things you'll need downloaded before you start
### Unity 2022.3.22f1
VRChat's Unity version, will be needed for using the VRChat SDK
[Found Here](https://unity.com/releases/editor/whats-new/2022.3.22f1 "https://unity.com/releases/editor/whats-new/2022.3.22f1")
### VRChat Creator Companion
You will be managing all your unity projects here any updates to the SDK and such you can update here
https://vrchat.com/home/download
![[Pasted image 20260623150913.png]]

Run though the process of the installer until you get to the main page (hit next repeatedly)

If you had installed the unity version above beforehand, it should automatically scan it and add it to the project, if not, you will need to do that first and manually select the folder it was installed to.

# Getting Started

When you're done with the process, you should land on a screen such as this 

![[Pasted image 20260623152420.png]]

anything not covered in this guide, you should be able to use the official VRChat to learn, and if not that, YouTube or google will be your best friend.

To get started, click on Projects on the left side of the computer, this can be found here

![[Pasted image 20260623152707.png]]

Go ahead and click on new project and you'll see a screen such as this. ==**Click on Unity 2022 World Project**== and go ahead and give the project a name.

![[Pasted image 20260623152813.png|524]]

When you click create, you should see a page similar to this. To keep this simplistic, we will just keep the default addons. Click on the big ah BLUE button that says "Open Project"

![[Pasted image 20260623152949.png]]

Once the loading screen is done you should see something like this

![[Pasted image 20260623153852.png]]

From here, the important things to note are

Left window is your Hierarchy, anything you create will be added to the side here

![[Pasted image 20260623154024.png]]

to the bottom is your Project Assets, anything here will be files that are in your project (IE textures or any fbx or such)

![[Pasted image 20260623154106.png]]

When you click on something, the right side should come to life, this is your inspector, it will show you details you can mess around with relating to the item you've clicked on (item will be highlighted in orange at the center)

![[Pasted image 20260623154119.png]]

If you want to play around with it a bit, check this window out to your top left, it will let you create shapes in unity you can move around. Try making a house or something out of it to get practice. Familiarize yourself with the controls and where to find things, it will help.

![[Pasted image 20260623154330.png]]

This concludes the basic steps for setting up a unity project.
